### Task Manager

- app.get('api/v1/tasks')  - get all the tasks
- app.get('api/v1/tasks/:id') - get a task by id
- app.post('api/v1/tasks') - create a task
- app.patch('api/v1/tasks/:id') - update a task
- app.delete('api/v1/tasks/:id') - delete a task
